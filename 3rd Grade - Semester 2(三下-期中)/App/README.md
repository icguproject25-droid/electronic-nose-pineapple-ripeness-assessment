🍍 鳳梨熟度檢測系統 (Pineapple Ripeness Detection System)
本專案是一款結合 AI 感測技術與行動裝置的應用程式，旨在透過電子鼻技術辨識鳳梨熟度。系統區分為消費端與農民端雙版本介面，滿足不同情境下的檢測與管理需求。
----------------------------------------------------------------------------------

🚀 快速啟動指南
請確保你的電腦已安裝 Node.js。

1. 進入專案目錄
打開終端機（CMD 或 Terminal），進入 expo 資料夾：

Bash
cd expo
2. 安裝依賴套件
由於專案包含多種 React 套件，請使用以下指令以確保版本相容性：

Bash
npm install --legacy-peer-deps
3. 啟動開發伺服器
執行 Expo 啟動指令：

Bash
npx expo start
4. 預覽方式
手機預覽： 在手機安裝 Expo Go App，並掃描終端機顯示的 QR Code。

網頁預覽： 在終端機按下 w 鍵開啟瀏覽器模式。

----------------------------------------------------------------------------------

📱 系統架構與功能
本系統採雙端設計，分別對應一般消費者與農業生產者：

A. 消費端 App (Consumer Side)
重點：直覺、互動、知識傳遞。

首頁 (Home)： 快速啟動掃描，支援中英雙語即時切換。

掃描流程： 配合動畫與語音提示，引導使用者完成 30 秒感測。

熟度結果： 以顏色碼（綠/黃/橘/紅）直觀顯示熟度、信心值及食用建議。

歷史紀錄： 查閱過往單筆檢測清單。

鳳梨知識庫： 提供選購與保存的教育內容。

B. 農民端 App (Farmer Side)
重點：效率、批次管理、統計分析。

批次管理： 支援建立批次編號，進行多顆鳳梨連續掃描。

結果彙整： 以圓餅圖與表格呈現整批鳳梨的熟度分佈與品質摘要。

報表分析： 生成統計趨勢圖表，支援 PDF 匯出以利決策參考。

歷史管理： 依日期篩選與管理過往大規模掃描紀錄。

----------------------------------------------------------------------------------

🎨 UI 設計原則
農民友善設計： 採用大按鈕、高對比配色與大字體，適應戶外強光與手套操作。

圖像化辨識： 使用鳳梨專屬色標（綠=未熟、黃=初熟、橙=完熟、紅=過熟），降低認知負擔。

低學習成本： 流程單一化，避免複雜子選單，確保初次使用即可上手。

全系統雙語： 支援中/英一鍵切換，包含 UI 文字與語音提示。
